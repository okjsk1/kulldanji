시작페이지 설정 안내

브라우저 정책상 사이트가 자동으로 시작페이지를 설정할 수는 없어요.

  • Chrome: 설정 → 시작 그룹 → 특정 페이지 또는 페이지 집합 열기 → 이 주소 추가
  • Edge: 설정 → 시작, 홈 및 새 탭 → 시작 페이지 → URL 추가
  • Safari: 설정 → 일반 → 홈페이지 → 이 주소 입력
